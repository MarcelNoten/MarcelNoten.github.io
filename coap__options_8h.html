◆ CoAP_CheckForUnknownCriticalOption()

uint16_t CoAP_CheckForUnknownCriticalOption ( CoAP_option_t *  pOptionsListBegin)

Critical Option.

An option that would need to be understood by the endpoint ultimately receiving the message in order to properly process the message (Section 5.4.1). Note that the implementation of critical options is, as the name "Option" implies, generally optional: unsupported critical options lead to an error response or summary rejection of the message.

Parameters
CoAP_option_t*pOptionsListBegin

This function packs multiple CoAP options.

This function packs multiple CoAP options to the format specified at section 3.1 in RFC7252. The option array gets sorted by option numbers first and is than packed into the compressed byte array format with its delta encoding
