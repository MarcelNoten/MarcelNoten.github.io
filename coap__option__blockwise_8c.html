Coap set payload.

Set payload in resource handlers

payloadIsVolatile: Set to true for volatile memory. If false, pPayload MUST point to static memory that is not freed before the interaction ends which is hard to detect.

Parameters
CoAP_Message_t*pMsgReq: The request message
uint8_t*pPayload: The payload to be send
uint16_tpayloadTotalSize: The size of pPayload
boolpayloadIsVolatile: Set to true for volatile memory.
Returns
